# 合同会社春秋 ウェブサイト

合同会社春秋（税理士事務所）の公式コーポレートサイトです。

## 構成ファイル

| ファイル | 内容 |
|---|---|
| `index.html` | サイト本体（全セクション） |
| `style.css` | スタイルシート（和モダン・春秋カラー） |
| `script.js` | インタラクション（ハンバーガー、スクロール演出、フォーム） |
| `春秋Logo背景透過.png` | ロゴ画像 |

## セクション構成

1. **Hero** — キャッチコピー・CTA
2. **About** — 社名の由来・想い・バリュー
3. **Service** — 確定申告 / 記帳代行 / 起業サポート / 節税相談
4. **Message** — 代表挨拶（秋谷 栄子）
5. **Contact** — お問い合わせフォーム

## デザインコンセプト

- **テーマカラー**：秋色（琥珀 `#b8712e`）× 春色（桜 `#c8907a`）× 深藍（`#2e3c4f`）
- **フォント**：Noto Serif JP（見出し）× Noto Sans JP（本文）
- **雰囲気**：和モダン・落ち着き・やわらかさ・信頼感

## 備考

- 電話番号・メールアドレスは準備中のため未掲載
- 代表者写真はプレースホルダー（写真入手後に差し替え）
- お問い合わせフォームの送信は現在モック（バックエンド連携は別途対応）
